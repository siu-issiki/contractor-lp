## PROBLEM SECTION（課題提起）

**Section Title:** こんな課題、抱えていませんか？

**Pain Point 1:**
**見積もりに何週間も待たされる**
開発会社に問い合わせても、ヒアリング→要件整理→見積もり作成で数週間。その間にビジネスチャンスを逃していませんか？

**Pain Point 2:**
**開発費が高すぎて、踏み出せない**
大手に依頼すれば安心だけど、中間マージンや管理コストが積み重なり、見積もりを見て愕然。限られた予算では到底手が出ない。

**Pain Point 3:**
**納品まで何ヶ月もかかる**
開発がスタートしても、仕様変更のたびにスケジュールが延び、いつ完成するのか分からない。市場投入のタイミングを逃してしまう。

---

## AGITATE SECTION（課題の深掘り）

**Section Title:** その「当たり前」、もう古いかもしれません。

従来のシステム開発は、多くの人員を投入し、長い工程を経て完成させる「労働集約型」のビジネスモデルでした。

プロジェクトマネージャー、設計者、複数のエンジニア、テスター。関わる人数が増えるほど、コミュニケーションコストが膨らみ、それが見積もりに上乗せされます。

しかし今、AI開発ツールの進化により、少数精鋭のチームでも大規模な開発が可能になりました。必要なのは、AIを使いこなせるエンジニアと、正しい設計力です。

---

## SOLUTION SECTION（解決策）

**Section Title:** Antaresが、開発の常識を変える。

**Feature 1: 質問に答えるだけで即時見積もり**
開発したいシステムの種類や規模を選ぶだけ。複雑なヒアリングは不要です。簡単な質問に答えていくだけで、概算見積もりをお届けします。「まず相場を知りたい」という段階から、お気軽にどうぞ。

**Feature 2: AI活用で工数を大幅圧縮**
コード生成、テスト自動化、ドキュメント作成。あらゆる工程でAIを活用し、従来の開発では考えられなかったスピードで高品質なプロダクトを届けます。人件費・管理コストを構造的に削減するため、低価格を実現しています。

**Feature 3: フルスタック対応**
Webアプリ、モバイルアプリ、API、インフラ構築まで。フロントエンドからバックエンド、デプロイ環境まで一気通貫で対応。窓口が一つで済むため、コミュニケーションコストも最小限に。

**Feature 4: スタートアップを知るチーム**
私たちは、限られたリソースで最大の成果を出す必要があるスタートアップの現場を熟知しています。「まず動くものを素早く」「MVP検証から段階的に拡張」といった、スタートアップに最適な開発アプローチを提案します。

## HOW IT WORKS（ご利用の流れ）

**Section Title:** 見積もりから開発まで、シンプルに。

**Step 1: 簡単な質問に回答する**
開発したいシステムの種類や規模など、いくつかの質問に答えるだけ。専門知識は一切不要です。

**Step 2: 詳細ヒアリング**
概算に納得いただけたら、担当エンジニアが詳細をヒアリング。技術選定と開発計画を策定し、正式な見積もりをお出しします。

**Step 3: 開発スタート**
合意後、すぐに開発を開始。AIを活用した高速開発で、従来よりも大幅に短い期間で納品します。進捗はリアルタイムで共有します。

**Step 4: 納品・運用サポート**
完成したプロダクトを納品。ご要望に応じて、運用・保守サポートや機能追加にも対応します。

## FAQ

**Section Title:** よくあるご質問

**Q: AIで開発して品質は大丈夫ですか？**
A: AIはあくまで開発を効率化するツールです。設計・コードレビュー・テストはすべて経験豊富なエンジニアが責任を持って行います。AIの活用により、むしろ人的ミスを減らし、品質を向上させることができます。

**Q: どのような技術スタックに対応していますか？**
A: React / Next.js / TypeScript / Node.js / Python / Go / AWS / GCP など、モダンな技術スタックに幅広く対応しています。お客様の要件に最適な技術を提案します。

**Q: 途中で仕様変更はできますか？**
A: はい。アジャイルな開発手法を採用しているため、開発途中での仕様変更にも柔軟に対応できます。AI活用により、変更に伴う手戻りコストも最小限に抑えられます。

**Q: 開発後の保守・運用もお願いできますか？**
A: はい。月額の保守・運用プランをご用意しています。障害対応、セキュリティアップデート、機能追加など、ご要望に応じて柔軟にサポートします。

**Q: NDA（秘密保持契約）は締結できますか？**
A: もちろんです。ご要望に応じて、プロジェクト開始前にNDAを締結いたします。

**Q: 見積もりだけでも大丈夫ですか？**
A: もちろんです。概算の確認は完全無料です。「まず相場感を知りたい」という方も、お気軽にご利用ください。
